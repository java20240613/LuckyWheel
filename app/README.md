# 🎡 LuckyWheel - Android 抽獎轉盤

一個使用 Android Custom View + Canvas 製作的抽獎轉盤 App。

---

## 🚀 功能特色

- 🎡 自訂轉盤（Canvas 繪製）
- 🎯 隨機抽獎機制
- ✨ 轉盤動畫（ObjectAnimator）
- 🔊 震動回饋（Vibrator）
- 📝 可動態新增選項
- 🎨 彩色分段 UI

---

## 🧠 技術重點

- Custom View（WheelView）
- Canvas drawArc / drawText
- ObjectAnimator 動畫控制
- Math.sin / cos 座標計算
- RecyclerView 管理列表
- Android Vibrator API
